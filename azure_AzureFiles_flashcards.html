Azure Flashcards: Azure Files

Test your knowledge about Azure Files with these flashcards. Click on a card to reveal the answer.

What is Azure Files?

Fully managed file shares in the cloud accessible via SMB, NFS, and REST API protocols.

Which OS can access SMB Azure file shares?

Windows, Linux, and macOS clients.

What is Azure File Sync?

Caches SMB Azure file shares on Windows servers for fast access.

Common use case for Azure Files?

Replacing or supplementing on-premises file servers and NAS devices.

How does Azure Files support "lift and shift"?

Enables applications expecting a file share to store data to be moved to the cloud.

How does Azure Files simplify cloud development?

Provides shared application settings, diagnostic shares, and tools for Dev/Test/Debug.

Which protocols are supported by Azure Files?

SMB and NFS protocols.

Recommended OS versions for SMB 3.1.1?

Windows 11 (21H2+), Windows 10 (21H1+), Windows Server 2019+, Linux kernel 5.3+.

Types of storage accounts used for Azure Files?

General purpose version 2 (GPv2) and FileStorage accounts.

How does Azure Files handle data redundancy?

Supports LRS, ZRS, GRS, and GZRS redundancy options.

Key features of Azure File Sync?

Centralizes file shares in Azure Files, caches them on-premises, supports cloud tiering.

What is SMB Multichannel?

Allows an SMB client to establish multiple network connections to an SMB file share.

How does Azure Files ensure data encryption?

Data is encrypted at rest using SSE and in transit using SMB 3.x with encryption or HTTPS.

Maximum size supported by standard SMB file shares with GRS and GZRS?

Up to 100TiB capacity.

Purpose of soft delete in Azure Files?

Allows recovery of a file share after accidental deletion within a retention period.

Benefits of Azure Files' integration with Azure Backup?

Provides scheduling, retention of snapshots, and protection against ransomware and data loss.

Supported billing models for Azure Files?

Pay-as-you-go for standard file shares and provisioned capacity for premium file shares.

Identity-based authentication methods for Azure Files?

Supports on-premises AD DS, Microsoft Entra Domain Services, and Microsoft Entra Kerberos for hybrid identities.

Available storage tiers for Azure Files?

Premium (SSD), transaction optimized, hot, and cool (HDD).

SLA for Azure Files Premium shares?

Azure Files Premium shares offer a 99.99% SLA.
