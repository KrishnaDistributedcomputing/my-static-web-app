Azure Flashcards: Azure Update Manager

Test your knowledge about Azure Update Manager with these flashcards. Click on a card to reveal the answer.

What is Azure Update Manager?

Azure Update Manager is a unified service to manage and govern updates for all machines across Azure, on-premises, and other cloud platforms from a single dashboard.

What are the key benefits of using Azure Update Manager?

Monitoring update compliance, deploying critical updates, flexible patching options, and a native experience without Azure Automation or Log Analytics dependencies.

What types of updates can Azure Update Manager handle?

Azure Update Manager handles operating system updates for both Windows and Linux machines.

What is the purpose of the Update Manager extension?

The extension interacts with the operating system to manage the assessment and application of updates, automatically installed during Update Manager operations.

What feature allows scheduling updates for Azure machines?

The "Schedule recurring updates" feature in Azure Update Manager allows scheduling updates for Azure machines.

How does Azure Update Manager ensure machine security?

Azure Update Manager secures machines by supporting automatic VM guest patching, hotpatching, and custom maintenance schedules.

What is the role of the Azure VM Windows agent in Azure Update Manager?

The Azure VM Windows agent installs and manages the Update Manager extension for Azure VMs.

What are the prerequisites for using Azure Update Manager?

Prerequisites include proper role assignments like Azure Virtual Machine Contributor or Azure Connected Machine Resource Administrator and ensuring the target machines meet specific OS requirements.

What regions support Azure Update Manager?

Azure Update Manager is supported globally in all Azure compute and Azure Arc regions.

How is update compliance data stored in Azure Update Manager?

Update compliance data is stored in Azure Resource Graph.

What happens when the Azure Log Analytics agent is retired?

The Azure Automation Update Management solution relying on this agent will encounter issues, so users are encouraged to move to Azure Update Manager.

What is Dynamic Scoping in Azure Update Manager?

Dynamic Scoping is an advanced capability that allows creating groups of machines based on a schedule to apply patches at scale.

Can Azure Update Manager manage updates for custom images?

Yes, Azure Update Manager supports managing updates for VMs created from customized images.

What update sources are supported by Azure Update Manager?

Supported update sources include Microsoft Update, Windows Server Update Services (WSUS), and public or local YUM or APT repositories for Linux.

How does Azure Update Manager handle periodic assessments?

Azure Update Manager can automatically assess machines for pending updates every 24 hours using periodic assessment.

What are the supported operating systems for Azure Update Manager?

Supported operating systems include various versions of Windows Server and Linux distributions such as Oracle Linux, Red Hat Enterprise, SUSE Linux, and Ubuntu.

What does the term "hotpatching" refer to in Azure Update Manager?

Hotpatching refers to a method of patching that reduces the need for reboots by applying updates without restarting the machine.

What is the pricing for Azure Update Manager on Arc-enabled servers?

Azure Update Manager is charged at $5 per server per month for Arc-enabled servers.

What feature does Azure Update Manager provide for managing update schedules at scale?

Azure Update Manager provides the "Schedule patching" feature for managing update schedules at scale using Azure Policy.

Can Azure Update Manager integrate with Azure Lighthouse?

Currently, Azure Update Manager does not support integration with Azure Lighthouse.
