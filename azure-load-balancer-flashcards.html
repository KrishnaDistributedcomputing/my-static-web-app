Azure Load Balancer Flashcards

Test your knowledge about Azure Load Balancer with these flashcards. Click on a card to reveal the answer.

What is Azure Load Balancer?

Azure Load Balancer distributes inbound flows arriving at the load balancer's front end to backend pool instances according to configured load-balancing rules and health probes.

What are the types of Azure Load Balancers?

Public Load Balancer and Internal (Private) Load Balancer.

What does the Public Load Balancer provide?

It provides outbound connections for VMs inside your virtual network by translating their private IP addresses to public IP addresses.

What is the role of an Internal Load Balancer?

It is used to load balance traffic inside a virtual network or in a hybrid scenario where the frontend IP is accessed from an on-premises network.

What are some key scenarios for using Azure Standard Load Balancer?

Load balancing internal and external traffic to Azure VMs, configuring outbound connectivity for VMs, using health probes, and enabling support for IPv6.

What are health probes in Azure Load Balancer?

Health probes monitor the status of your app to determine which instances are healthy and can receive traffic.

What is the difference between Basic and Standard Load Balancers?

Standard Load Balancer provides more features, better security, and higher scalability compared to Basic Load Balancer.

How does Azure Load Balancer handle availability zones?

Azure Load Balancer supports availability zones by distributing traffic across VMs in different zones, enhancing resilience and availability.

What are the pricing tiers for Azure Load Balancer?

Basic Load Balancer is free, while Standard Load Balancer has a cost associated based on the number of rules and data processed.

What is a NAT gateway in Azure Load Balancer?

A NAT gateway provides outbound Internet connectivity for one or more subnets of a virtual network.

How do you create a public IP address for a load balancer in Azure CLI?

Use `az network public-ip create` command with appropriate parameters for resource group, name, SKU, and zone.

How does Azure Load Balancer support multiple frontends?

Azure Load Balancer can have multiple frontend IP configurations to load balance traffic on different IP addresses and ports.

What is the command to create a health probe in Azure CLI?

Use `az network lb probe create` command with parameters for resource group, load balancer name, probe name, protocol, and port.

What is the purpose of a backend pool in Azure Load Balancer?

A backend pool contains the virtual machines that the load balancer distributes traffic to.

What does the term 'Idle timeout' refer to in Azure Load Balancer?

'Idle timeout' refers to the duration after which an idle connection is closed by the load balancer.

How can you test the load balancer setup in Azure?

Copy the public IP address of the load balancer and paste it into a browser to check the default page of the IIS web server on backend VMs.

What is a load balancing rule in Azure Load Balancer?

A load balancing rule defines how traffic is distributed to the VMs in the backend pool, specifying frontend IP, backend pool, protocol, and ports.

What is the function of 'Floating IP' in Azure Load Balancer?

'Floating IP' enables direct server return, allowing traffic to be routed directly back to the client without passing through the load balancer.

What is the purpose of Network Security Groups (NSGs) in Azure Load Balancer?

NSGs are used to control inbound and outbound traffic to the resources in the virtual network, enhancing security.

How can you create a load balancer in Azure PowerShell?

Use the `New-AzLoadBalancer` cmdlet with parameters for resource group, name, location, SKU, frontend IP configuration, backend pool, and health probe.
