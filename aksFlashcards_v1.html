Managed control plane

Azure manages control plane components like API server, etcd, and controller manager.

Scalability

Supports horizontal scaling through HPA, vertical scaling through VPA, and KEDA for event-driven scaling.

High availability

Supports multi-zone clusters and self-healing mechanisms to replace failed nodes.

Security

Offers built-in network security with Azure Policy and supports private clusters.

Monitoring and logging

Integrate with Azure Monitor to gain insights into your clusters.

Hybrid capabilities

Azure Arc-enabled Kubernetes extends AKS capabilities to any infrastructure.

Serverless Kubernetes

Use Azure Container Instances (ACI) as a virtual node to quickly scale your applications.

DevOps integration

Supports integration with CI/CD tools like Azure DevOps, Jenkins, and GitHub Actions.

Cost optimization

Spot instances reduce costs by using unused Azure capacity.

Global reach

Azure's global infrastructure allows you to deploy applications closer to users.

Open-source compatibility

Supports Helm, Prometheus, Istio, and many other open-source projects.

Windows container support

Supports both Linux and Windows containers in a single Kubernetes cluster.

GPU support

Supports NVIDIA GPUs for tasks like deep learning and simulations.

Private clusters

Ensures control plane access is restricted to your virtual network.

Azure policy integration

Define and enforce rules across your resources for compliance.

Azure backup integration

Provides a secure solution to back up and restore AKS workloads.

Azure monitor for containers

Provides detailed insights into containerized applications.

Azure Kubernetes fleet manager

Centralized monitoring and policy enforcement for multiple AKS clusters.

Artifact streaming

Improves speed and efficiency of deploying large container images.

Confidential containers

Use trusted execution environments (TEEs) to protect data.

Deployment safeguards

Ensure deployments adhere to best practices and reduce misconfigurations.

Azure CNI overlay

Enhances network scalability and performance using an overlay network.

Azure service mesh

Provides infrastructure for managing service-to-service communication with features like load balancing and mutual TLS.

Azure Arc-enabled Kubernetes

Manage, monitor, and secure Kubernetes clusters on-premises, at the edge, or in other clouds.
