Azure Flashcards: VPN Gateway

Test your knowledge about Azure VPN Gateway with these flashcards. Click on a card to reveal the answer.

What is Azure VPN Gateway?

A service for encrypted traffic between Azure VNets and on-premises locations.

What is Site-to-Site (S2S) VPN?

Connects on-premises network to Azure VNet using IPsec/IKE VPN tunnel.

What is Point-to-Site (P2S) VPN?

Connects individual client computers to Azure VNet over SSTP, IKEv2, or OpenVPN protocols.

What is VNet-to-VNet VPN?

Connects Azure VNets to each other using IPsec/IKE VPN tunnels.

What does Gateway SKU define?

Defines the VPN Gateway's capabilities, including number of tunnels, throughput, and features.

What is a Gateway Subnet?

A dedicated subnet for VPN Gateway within your VNet.

What is BGP?

BGP - Border Gateway protocol.Enables dynamic routing by exchanging routing information between Azure VPN Gateway and on-premises routers.

What is Active-Active Mode?

A high-availability configuration using two VPN Gateways in an active-active setup.

What is IPsec/IKE VPN?

Protocols for establishing secure VPN tunnels. IPsec provides encryption and IKE negotiates security associations.

What is ExpressRoute?

Provides a dedicated private connection to Azure, bypassing the public internet.

What is Route-Based VPN?

Uses dynamic routing protocols to manage network traffic.

What is Policy-Based VPN?

Uses static routing configurations, ideal for simpler, smaller networks.

What is a Shared Key?

A secret key used for establishing VPN connections.

What is IKEv2 Protocol?

A secure and efficient VPN protocol supported by Azure VPN Gateway.

What is OpenVPN Protocol?

An open-source, secure VPN protocol supported by Azure VPN Gateway.

What is a Gateway Public IP?

A static public IP address assigned to the VPN Gateway for external connections.

What is a Local Network Gateway?

Represents the on-premises location in Azure, including the public IP address of the VPN device and the on-premises address space.

What does Gateway Configuration involve?

Setting up VPN types, choosing SKUs, configuring BGP, and other parameters.

What is a Network Security Group (NSG)?

Not supported on the Gateway Subnet as it can disrupt VPN and ExpressRoute gateway functionality.
