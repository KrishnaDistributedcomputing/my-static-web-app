Azure Flashcards: Azure Files

Test your knowledge about Azure Files with these flashcards. Click on a card to reveal the answer.

What protocols does Azure Files support for accessing file shares?

Azure Files supports SMB, NFS, and Azure Files REST API.

What are some common use cases for Azure Files?

Common use cases include replacing on-premises file servers, cloud applications, and dev/test environments.

How does Azure File Sync enhance the performance of SMB Azure file shares?

Azure File Sync caches frequently accessed data on-premises for faster access.

What is the recommended OS for using NFS Azure file shares?

Recommended OS are Linux with kernel version 4.3+ and 5.3+ for enhanced features.

What is the difference between LRS and GRS?

LRS stores three copies of data within one region; GRS replicates data to a secondary region.

How does Azure Files ensure data encryption at rest?

All data is encrypted using Azure storage service encryption (SSE).

What are the benefits of using Premium file shares over Standard file shares?

Premium file shares offer SSD storage, providing low latency and higher IOPS.

What is the primary use of the Azure Files REST API?

It is used for programmatically managing Azure file shares.

How does Azure Files handle network isolation for security?

Azure Files can be accessed via private or service endpoints.

What is the maximum number of snapshots allowed per Azure file share?

Up to 200 snapshots per file share, with retention up to 10 years.

What are the benefits of using Azure Files in a hybrid environment?

Azure Files supports hybrid deployments and on-premises caching with Azure File Sync.

What are the available redundancy options for Azure Files?

Options include LRS, ZRS, GRS, and GZRS.

How does Azure Files support identity-based authentication?

Azure Files supports AD DS, Microsoft Entra Domain Services, and Entra Kerberos.

What is the purpose of soft delete in Azure Files?

Soft delete allows recovery of accidentally deleted file shares.

How can you increase the performance of NFS Azure file shares?

Use the `nconnect` Linux mount option for more TCP connections.

What is the function of Azure Backup for Azure file shares?

Azure Backup provides scheduled, policy-driven backups, protecting data from ransomware and data loss.

What is the difference between GPv2 and FileStorage storage accounts?

GPv2 hosts multiple storage resources, while FileStorage specializes in premium SSD-based file shares.

How does Azure Files support Active Directory integration?

By domain-joining storage accounts to on-premises AD DS or Microsoft Entra Domain Services.

What are the supported encryption algorithms for SMB channel encryption?

Supported algorithms include AES-256-GCM, AES-128-GCM, and AES-128-CCM.

How does Azure Files handle disaster recovery and failover for file shares?

Azure Files supports GRS and GZRS, allowing failover to a secondary region in case of a disaster.
