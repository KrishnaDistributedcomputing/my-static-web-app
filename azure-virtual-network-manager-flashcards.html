Azure Virtual Network Manager Flashcards

Test your knowledge about Azure Virtual Network Manager with these flashcards. Click on a card to reveal the answer.

What is Azure Virtual Network Manager (AVNM)?

A management service to group, configure, deploy, and manage virtual networks globally across subscriptions.

How does AVNM handle connectivity configurations?

AVNM supports hub-and-spoke and mesh topologies for managing network connectivity configurations.

What is the primary benefit of AVNM's security admin rules?

Security admin rules have higher priority than NSG rules, providing centralized enforcement of security policies.

What does AVNM automate in network management?

AVNM automates the creation and management of virtual network peerings, reducing manual configuration.

How does AVNM improve network performance?

By enabling direct connectivity between spokes and optimizing traffic flows, AVNM reduces network latency.

Can AVNM manage virtual networks across regions and subscriptions?

Yes, AVNM centrally manages and configures connectivity policies across different Azure regions and subscriptions.

What is a hub-and-spoke topology?

A network topology where a central hub virtual network connects to multiple spoke virtual networks.

What does AVNM's mesh topology configuration allow?

It allows all virtual networks to communicate directly with each other without routing through other networks.

How does AVNM enforce security policies?

By creating security admin rules that apply to all virtual networks, ensuring a baseline level of security.

What is the benefit of AVNM's automated network topology maintenance?

It ensures consistent connectivity even as new virtual networks are added or removed.

What is the purpose of network groups in AVNM?

Network groups logically segment virtual networks to apply specific configurations and policies.

What feature of AVNM helps reduce network latency?

Enabling direct connectivity between spoke networks without routing through the hub.

How does AVNM support security for critical services?

By creating "force-allow" rules to prevent accidental blocking of essential traffic.

How can AVNM help with enforcing organizational security policies?

AVNM can create security rules that cannot be overridden at the individual virtual network level.

What does AVNM provide in terms of centralized management?

It allows centralized management and configuration of network connectivity and security policies globally.

What does AVNM's dynamic membership feature use?

It uses Azure Policy to dynamically add virtual networks to network groups based on specified conditions.

How does AVNM reduce the risk of manual errors?

By automating the deployment and configuration of network settings across virtual networks.

What is the key difference between security admin rules and NSG rules in AVNM?

Security admin rules have higher priority and are evaluated first, overriding NSG rules.

What flexibility does AVNM provide for VNet owners?

VNet owners can configure network security groups (NSGs) as needed for more specific traffic management.

How does AVNM help with monitoring and diagnostics?

AVNM integrates with monitoring tools to quickly identify and resolve network issues.
