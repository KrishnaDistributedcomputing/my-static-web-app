Azure Flashcards: Virtual Desktop

Test your knowledge about Azure Virtual Desktop with these flashcards. Click on a card to reveal the answer.

What is Azure Virtual Desktop (AVD)?

AVD is a desktop and app virtualization service that runs on Azure, providing a full Windows experience with options for multi-session or single-session usage.

What operating systems does Azure Virtual Desktop support?

AVD supports Windows 11 Enterprise multi-session, Windows 11 Enterprise, Windows 10 Enterprise multi-session, Windows 10 Enterprise, Windows Server 2022, Windows Server 2019, and Windows Server 2016.

What are the two types of host pools in Azure Virtual Desktop?

The two types of host pools are Personal host pools and Pooled host pools.

What is the main benefit of using pooled host pools in Azure Virtual Desktop?

Pooled host pools provide a shared remote experience to end-users, ensuring lower costs and greater efficiency.

How does Azure Virtual Desktop manage session hosts and virtual machines?

Azure manages the infrastructure roles like gateway and broker, while the customer manages session hosts, virtual network connectivity, resiliency, and backup.

What is the purpose of the Azure Virtual Desktop Insights?

AVD Insights provide key metrics and analytics about the environment and user connections, helping in troubleshooting and optimization.

What are the supported identity providers for Azure Virtual Desktop?

Supported identity providers include Microsoft Entra ID, Active Directory Domain Services (AD DS), and Microsoft Entra Domain Services.

What are the key capabilities of Azure Virtual Desktop?

Key capabilities include multi-session Windows experience, pooled resources to reduce costs, auto-scaling, and integration with Microsoft 365 apps.

What is FSLogix used for in Azure Virtual Desktop?

FSLogix is used for profile management, allowing user profiles to be stored on Azure Files or Azure NetApp Files for flexible and persistent user data.

What role does Azure Traffic Manager play in Azure Virtual Desktop?

Azure Traffic Manager routes user devices to the closest instance of the AVD web service based on geographic traffic-routing methods.

What is the purpose of the Remote Desktop Protocol (RDP) Shortpath in Azure Virtual Desktop?

RDP Shortpath improves user experience by establishing a direct UDP-based transport between the user's device and the session host, bypassing the gateway service.

What are the Azure Virtual Desktop licensing options for internal users?

Internal users can use Microsoft 365 E3/E5/A3/A5, Windows Enterprise E3/E5, Windows Education A3/A5, or Windows VDA per user.

How does Azure Virtual Desktop handle application publishing?

Applications can be published individually using RemoteApp or as part of a full desktop experience from a host pool.

What is the role of Azure Front Door in Azure Virtual Desktop?

Azure Front Door provides the global entry point to AVD, directing user connections to the appropriate gateway service based on latency.

What is an application group in Azure Virtual Desktop?

An application group is a logical grouping of applications available on session hosts within a host pool, controlling user access to full desktops or individual apps.

What is the purpose of a validation environment in Azure Virtual Desktop?

A validation environment allows testing of service updates before they are applied to production environments, helping to identify potential issues.

What are the costs associated with Azure Virtual Desktop?

Costs include Azure resource consumption (VMs, storage, network), licensing for Microsoft Entra ID or AD DS, and any additional features like AVD Insights or FSLogix.

What is the main use of autoscale in Azure Virtual Desktop?

Autoscale helps manage costs by automatically scaling session hosts based on capacity thresholds and schedules.

What is required for network connectivity in Azure Virtual Desktop?

A virtual network and subnet for session hosts, ability to connect to domain controllers, and unrestricted access to specific Azure URLs over TCP port 443.

What is the difference between active, disconnected, and pending user sessions in Azure Virtual Desktop?

Active sessions: signed in and connected users. Disconnected sessions: inactive without sign-out. Pending sessions: placeholders reserving spots..
